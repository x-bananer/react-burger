# Burger Restaurant SPA

A fun, space-themed front-end single-page application for a burger restaurant with menu browsing and order flow. Built with **React**, **TypeScript**, and **Vite**. Focus on modular components and maintainable UI logic.

![App preview](/public/preview.png)

## Features
- Menu browsing
- Ingredient composition (drag & drop)
- Order flow with real-time status updates (WebSockets)
- Loading and error states
- Responsive layout

## Tech Stack
- React 18
- TypeScript
- Vite
- Redux Toolkit + Redux Thunk
- React Router
- React DnD
- WebSockets (real-time updates)

## Requirements
- Node.js 20.19.2
- npm 10.8.2

## Setup
Install dependencies:
```bash
npm install
```

Run the app:
```bash
npm run dev
```

## Scripts
- `npm run dev` - start dev server
- `npm run lint` - run ESLint

## Notes
This project demonstrates component architecture, typed data flow, and maintainable UI structure.
